How it works

This tool uses direct PCM encoding to convert any file data into audio samples. Two bytes are packed into each 16-bit audio sample, making the audio file the same size as your original file. All processing happens in your browser - no data is uploaded to any server.

  • Works with ANY file type (images, documents, videos, archives, etc.)
  • Support for multiple files - combine many files into one audio
  • Optional AES-256-GCM encryption with password protection
  • Audio file size = Total files size + small header
  • Sample rate: 202.86 kHz (4.6x shorter audio than standard)
  • 100% efficient encoding (2 bytes per sample)
